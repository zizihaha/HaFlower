## 電商網站 : HA-flower
#### Demo : https://zizihaha.github.io/ec_flower/#/


## 使用技術
* Vue CLI
* Vue Router
* Axios
* API
* ESlint
* Bootstrap 4
* RWD
* SCSS


## 使用插件
* vue-router
* vue-axios
* vue-loading-overlay
* vue-lazyload
* swiper
* bootstrap 4
* Font Awesome

## 功能介紹
#### :sun_with_face: 前台
* 首頁
* 商品列表頁
* 單一商品頁面
* 可建立喜愛商品清單(存於localStorage)
* 購物車功能



#### :wrench: 後台
* 管理員登入頁面
* 可建立工作待辦備忘錄 (存於localStorage)
* 管理商品(可新增/編輯/刪除)
* 預先上架商品，可設定前台不顯示
* 瀏覽所有消費者訂單資訊
